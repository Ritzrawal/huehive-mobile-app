"Croma est un gestionnaire de palette de couleurs simple et un sélecteur de couleurs conçu pour les concepteurs, qui vise à rendre rapide et amusant la création et le partage de palettes de couleurs en déplacement.


PRINCIPALES CARACTÉRISTIQUES

- Extrayez automatiquement les couleurs saillantes d'une image provenant de votre galerie ou de votre appareil photo grâce à notre algorithme maison unique.
• Choisissez manuellement n'importe quel numéro de couleurs dans un objet avec votre appareil photo.
• Sélectionnez manuellement n'importe quel numéro de couleurs à partir d'images.
- Reconnaissez et enregistrez les codes de couleur hexagonaux dans le texte à l'aide de votre appareil photo.
- Générez des palettes avec différents modèles de couleurs tels que complémentaires, fractionné-complémentaires, triadiques, tétradiques, analogues, monochromatiques, etc.
- Partagez les palettes avec un lien pour un aperçu rapide.
• Affichez diverses informations sur une couleur et convertissez en d'autres codes de couleur.
• Copiez les codes de couleur dans votre presse-papiers en un clic.
• Ajoutez des couleurs à l'aide de divers formats de code de couleur, y compris les couleurs nommées CSS.
• Palettes de conception de matériaux



AVANTAGES PRO

• Ajoutez plus de 4 couleurs dans une palette.
• Exportez/synchronisez les palettes vers le référentiel privé GitHub


DESCRIPTION

Extrayez automatiquement les couleurs dominantes d'une image provenant de votre galerie ou de votre appareil photo grâce à notre algorithme maison conçu pour les palettes de couleurs, ou choisissez manuellement un nombre quelconque de couleurs avec le sélecteur de couleurs dédié de votre appareil photo.


Récupérez les codes de couleur de n'importe quel texte aléatoire, comme un courriel envoyé par votre client, ou des couleurs que vous trouvez dans un article d'une page Web. C'est simple et facile.


Générez des schémas de couleurs harmonieux à partir d'une couleur existante en utilisant divers modèles tels que complémentaire, split-complémentaire, triadique, tétradique, analogue, monochromatique, etc. et enregistrez-les dans votre bibliothèque.


Le partage est important, c'est pourquoi nous avons facilité plus que jamais le partage des palettes de couleurs. Le destinataire reçoit un lien pour prévisualiser rapidement les couleurs dans Croma. Même s'il n'a pas l'application.


Voir toutes sortes de détails sur une couleur. Convertissez facilement n'importe quel code couleur en différents formats de code couleur. Lors de l'ajout d'une couleur, vous pouvez utiliser n'importe quel format de code de couleur, y compris les couleurs nommées CSS. Copiez le code couleur dans votre presse-papiers d'un simple toucher.


Nous avons conçu Croma pour aider les concepteurs à gérer facilement les couleurs, car les couleurs comptent. Alors qu'il est très facile d'avoir beaucoup de fonctionnalités et d'être gonflé, nous aimons garder les choses simples, vous aider avec les choses dont vous avez le plus besoin et vous permettre de travailler de manière transparente sans vous gêner.


Croma peut être utile dans diverses tâches, telles que la conception Web et graphique, partout où les couleurs sont nécessaires. C'est un outil de productivité indispensable pour les concepteurs.
